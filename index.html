多元的课程体系

涵盖符合不同年龄、不同能力阶段的多元课程

Basic

初级启蒙

KOOV机器人编程启蒙

索尼国际教育推出的针对3-12岁孩子的编程启蒙课程，KOOV®是一款可以用模块自由拼搭出各种“造型”，通过“编程”赋予作品各式“动作”，让孩子们动手制作的过程中，培养孩子们的创造力，好奇心，以及开拓未来的思维能力。

英国 BBC Microbit 编程

Microbit 是由英国广播公司BBC推出的一款适用于STEM教育，方便小朋友们学习编程的微型控制板，采用Microbit可以学习到加速度传感器、蓝牙等多种传感器的应用，并且结合微软推出的Makecode编程平台，方便易学，轻松上手。

Arcade 游戏编程

喜欢游戏是孩子的天性，借助孩子对游戏的兴趣，学习开发孩子自己的游戏。使用微软 MakeCode Arcade 编程平台，孩子可以充分的发挥自己的创意，创作出属于自己的掌机游戏！快乐中学习，获得编程技能和思维，展现孩子无限的创造力！

Advanced

高阶、专业、实战

Python 编程基础

学习Python的语法基础，包括Python的数据类型、变量、循环、判断、函数、类等语法知识点。学习常用的数据类型的使用方式。

Python 游戏开发

Python开发实战，学习Turtle模块，利用Turtle模块学习游戏设计与开发，应用 Python 基础知识，通过逻辑组合完成游戏制作。

图像处理

图像处理是现代人工智能中非常重要的组成部分。本课程学习通过opencv进行图像的数字处理，实现类似Photoshop和美图秀秀的基础功能。

人脸识别

人脸识别是人工智能最广泛的应用场景，本课程学习人脸识别的基本原理，如何通过Python进行人脸识别应用的开发，学习本地人脸识别和云端人脸识别等不同方式的人脸识别实现方式。

自动驾驶

自动驾驶是人工智能最重要的应用领域，本课程学习车道线检测，模拟自动驾驶的实现算法。线下课学习制作自动驾驶的人工智能小车。

AI 互动游戏

交互设计是计算机科学中最终要的分支。索尼的Playstation、微软的xbox Kinect、任天堂的Switch，都适用到了人工智能交互，来增加游戏的趣味性。本课程学习制作基础的人工智能交互游戏。

GUI 软件开发

GUI全程Graphical User Interface，我们所使用的几乎所有软件都是GUI编程实现的。本章节学习使用Python的GUI开发，制作音乐播放器软件。

App 开发

本课程从零学习打造一款iOS App。课程将设计UI、UX基础的交互，学习界面设计，App功能实现，Swift编程语言的使用。最终开发出属于自己的iPhone/iPad App。

Fullstack 全栈工程师

全栈工程师是IT公司中能力非常强的软件工程师，具备独立完成整个产品的开发能力。包含了UI设计、交互设计、前端软件开发、后台服务器开发、业务理解和实现的综合能力。本课程面向以全栈工程师为目标的专业人员。
